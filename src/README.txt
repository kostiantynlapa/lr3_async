
Цей проект містить два завдання, які демонструють використання багатопотоковості за допомогою `Fork/Join Framework`.
Завдання 1: Сума елементів стовпців матриці
Опис:
Програма генерує матрицю заданого розміру з випадковими елементами в межах, визначених користувачем. Вона обчислює суму кожного стовпця матриці, використовуючи Fork/Join Framework для паралельної обробки.

Інструкція з виконання:
1. Запустіть програму `ColumnSumCalculator`.
2. Введіть розміри матриці (кількість рядків і стовпців).
3. Вкажіть початкове та кінцеве значення для генерації випадкових чисел.
4. Програма виведе:
   - Згенеровану матрицю.
   - Суми елементів кожного стовпця.
   - Час виконання програми.

Файли:
- `ColumnSumCalculator.java`: Основний код для першого завдання.


Завдання 2: Пошук файлів більшого розміру
Опис:
Програма шукає файли у вказаній користувачем директорії (включаючи піддиректорії), розмір яких перевищує вказане значення, використовуючи Fork/Join Framework.

Інструкція з виконання:
1. Запустіть програму `FileSizeFilter`.
2. Введіть шлях до директорії, в якій потрібно виконати пошук.
3. Вкажіть мінімальний розмір файлів у байтах.
4. Програма виведе:
   - Шлях до кожного знайденого файлу.
   - Розмір кожного знайденого файлу.
   - Повідомлення про завершення пошуку.

Файли:
- `FileSizeFilter.java`: Основний код для другого завдання.
